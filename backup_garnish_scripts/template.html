📋 Шаблон меню Gastroprime

Импорт недельного меню в приложение app.Gastroprime.ru

Как использовать шаблон

1

Скачайте шаблон

Нажмите кнопку "📥 Скачать шаблон TSV" ниже. Файл откроется в Excel или другом редакторе таблиц.

2

Заполните меню

Внесите названия блюд для каждого дня недели. Используйте точные названия из базы данных Gastroprime.

3

Сохраните как CSV

В Excel: Файл → Сохранить как → Выберите "CSV (разделители - запятые)" → Сохранить.

4

Импортируйте в приложение

В админке app.Gastroprime.ru перейдите в "Планирование" → "Импорт меню" → Загрузите CSV файл.
